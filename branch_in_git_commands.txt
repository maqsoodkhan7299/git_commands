create a new branch
git branch <branch-name> 
git branch second-branch

We can see the new branch with the name "second-branch", 
but the * beside master specifies that we are currently on that branch.
git branch
* main
  second-branch


"checkout is the command used to check out a branch. 
"Moving us from the current branch, to the one specified at the end of the command:

git checkout second-branch

"steps to add changes to git repository
0. change branch by: git checkout second-branch
1. git add --all 
2. git status #to check the status of the files
3. git commit -m "message"
4. git push -u origin second-branch
